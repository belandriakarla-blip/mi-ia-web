# IA Clasificadora de Números (MNIST) - Karla

Este proyecto es una aplicación web interactiva que utiliza una red neuronal convolucional para reconocer números dibujados a mano alzada.

## 🚀 Funcionalidades
- **Lienzo interactivo:** Dibuja cualquier número del 0 al 9.
- **Predicción en tiempo real:** La IA procesa la imagen y devuelve el resultado.
- **Gráfico de confianza:** Muestra la probabilidad que la IA asigna a cada número.

## 🛠️ Tecnologías utilizadas
- **Python** (Lenguaje principal)
- **TensorFlow/Keras** (Entrenamiento y carga del modelo)
- **Streamlit** (Interfaz web y despliegue)
- **OpenCV** (Procesamiento de imagen)

## 📁 Estructura del Repositorio
- `app.py`: Código de la interfaz de usuario.
- `requirements.txt`: Librerías necesarias para el funcionamiento.
- `modelo_mnist.keras`: Modelo de IA ya entrenado.
